# UniSync Functional Mobile App

Fully functional React Native Expo app with all backend features.

## Quick Start

1. Edit `app/services/api.ts` - Change IP address
2. Run `npm install`
3. Run `npm start`
4. Scan QR code with Expo Go

## Features

✅ Student & Society Login
✅ Dashboards with stats
✅ Create/Edit/Delete events
✅ Image upload
✅ Join/Leave events
✅ View participants
✅ Pull to refresh
✅ All backend endpoints integrated

## Usage

Find your IP: `ipconfig` (Windows) or `ifconfig` (Mac/Linux)
Update `API_BASE_URL` in `app/services/api.ts`
